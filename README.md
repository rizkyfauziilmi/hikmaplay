# HikmaPlay

**HikmaPlay** is a curated global Islamic video platform that offers a structured, interactive, and educational watching experience. It features YouTube videos curated with summaries, notes, and user progress tracking.

## Key Features

- Embed YouTube videos
- User progress tracking
- Markdown support for summaries, notes, and insights
- Video categories & tags
- Bookmark videos and create playlists
- Optional AI/manual curated content

## Target Users

- Global Muslim audience
- All ages interested in learning Islamic knowledge
- Users who want to watch while taking notes or reflecting
